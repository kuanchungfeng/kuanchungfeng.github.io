資料夾結構

  1. archetype

    • like a meta-data，產生預設的frontMatter

  2. content

    • 基本上所有內容都放在這裡，例如，你要寫一個部落格，文章都放在此資料夾。

    • 假如你要寫一個網頁，html檔也放在這裡面。

  3. data

    • 就像是資料庫存放位置，當然，hugo是產生靜態網頁，不會直接從資料庫連線獲取資訊，但你可以將json檔放在此資料夾，去顯示對應的資料。

  4. layouts

    • 放一些重複的模板，例如寫一個header.html 或 footer.html讓每頁都去呼叫這些模板

  5. static

    • 放image，css，js檔等等不會改變的檔案。

  6. theme

    • 可以使用別人寫好的樣板(主題)，去 官網主題 選擇一個你喜歡的hugo主題，git submodule到你的hugo 資料夾，下載的主體通常會放在此資料夾。

  7. config.toml

    • 像是yaml檔，設定你網頁的參數

Note
 不一定每個資料夾都會用到，有可能你只使用theme 跟 content 來完成簡單的blog

Content

Hugo 分成兩種頁面跟一種特殊的頁面：

  1. List Page

  2. Single Page

  3. _index.md

    • hugo 只會自動產生第一層的資料夾，不會產生第二層以下的資料夾

    • 如果要產生第二層以下的資料夾頁，可以加上_index.md到該層資料夾

    • _index.md 可以用來控制及增加一些特定的內容於List Page 之頂端的頁面。

frontMatter

  1. frontMatter是content裡面每篇內文的meta-data

  2. 格式可以為yaml,toml,json

  3. 預設格式為 yaml

yaml 用 三條 - 隔開，例如：

---
title: "A"
date: 2020-11-30T16:32:47+08:00
draft: true
---

toml 用 三個 + 隔開，例如：

+++
title =  "A"
date =  2020-11-30T16:32:47+08:00
draft =  true
+++

  1. frontMatter 可以讓你輕鬆展示檔案的名字、日期等檔案資訊。

  2. 可以自己客製你想要的forntMatter ，例如：

---
title: "A"
date: 2020-11-30T16:32:47+08:00
draft: true
Author: "Arthur"
---

  1. 利用frontMatter管理你的template，你可以想像成用不同標籤分類你的檔案，再用Hugo去做不同標籤相對應的排版。

If Statements






判斷式規則：{{ if operator param1 param2}}






operator 有






    

  • 

    eq
    

    • 

  • 

    lt(less than)
    

    • 

  • 

    le(less than and equal to)
    

    • 

  • 

    gt(greater than)
    

    • 

  • 

    ge(greater than equal to)
    

    • 

  • 

    not
    

    • 







例如：








  {{ $var1 := 1}}
  {{ $var2 := 2}}
  {{ if eq $var1 $var2}}
    true
  {{else}}
    false
  {{end}}








如果要用not，記得將條件包在()裡面，例如：








  {{ $var1 := 1}}
  {{ $var2 := 2}}
  {{ if not (eq $var1 $var2) }}
    true
  {{else}}
    false
  {{end}}








如果要兩個條去做and 或 or ，格式為{{ if and (cond1) (cond2) }}，例如：








  {{ if and (lt $var1 $var2) (gt $var2 $var1) }} //(1)
    true
  {{else}}
    false
  {{end}}








    

  1. 

    lt $var1 $var2 為 $var1 less than $var2
